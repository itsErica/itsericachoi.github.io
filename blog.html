How to Create a Website.

How to create and edit a website by using GitHub and Visual Studio Code(VSC), May 12, 2020

First, create a domain: such as, namecheap (free with student email) or porkbun.com (reasonable pricings)
On GitHub, create a repository as "blah.github.io"; blah, can be any related name for your site. This will be used as a host to save your website files and codes.
Open the GitHub repository that you created. Click on setting, scroll down to "GitHub Pages": have your source as "master branch", save cutom domain as your domain website, and enable enforce https (for added security).
Then, using Github to connect with your domain (porkbun.com): sign in. Under domain details, "quick connect" your github account then connect your domain.

Congratulations you have made a website and it is ready to be edited.

To edit a website with Visual Studio Code VSC (free coding software made by Microsoft):
On the left, click on "source control" button and it will tell you to install git.
After downloading GIT, restart Visual Studio Code, under Source control, clone your repository; go to your GitHub repository, Code > "clone or download" > copy clone link, then paste it into VSC.


Now you are ready to start editing your code!

Create an "index.html" file; this will be your home link.
After editing your code, save and then on the top menu bar, click terminal > new terminal and enter the following commands:
"git config --global user.email "you@email.com"
"git config --global user.name "githubusername"
This will let GIT system know who's saving the changes as.

When you are ready to upload your saves on to your github through VSC; on the left menu > source control > message "commit changes" and press ctrl+enter on your keyboard to push and commit changes.
